Pengenalan CSS (Cascading Style Sheets)

Memahami dasar-dasar dan penerapan CSS dalam pengembangan web

Tujuan Pembelajaran

  • Memahami tentang konsep CSS
  • Memahami aturan penulisan pada CSS
  • Memahami selector sebagai pengontrol desain
  • Memahami pembuatan background
  • Memformat teks pada web
  • Memahami pengaturan bentuk font

Konsep CSS

CSS (Cascading Style Sheets) adalah bahasa desain yang digunakan untuk mengatur tampilan dan format dokumen HTML.

Dengan CSS, kita bisa memisahkan antara struktur konten (HTML) dengan desain visual. CSS bekerja melalui selector dan declaration.

Setiap declaration dalam CSS terdiri dari dua bagian utama:

  • Property: Merupakan atribut atau hal yang ingin diatur, seperti color, font-size, atau background.
  • Value: Nilai yang diberikan kepada property, seperti red, 14px, atau center.

Contoh: color: red; → color adalah property, dan red adalah value.

Aturan Penulisan CSS

Contoh penulisan CSS dan HTML-nya:

p {
  color: red;
  font-size: 14px;
}

Contoh HTML:

<p>Ini adalah paragraf dengan teks merah dan ukuran 14px.</p>

Ini adalah paragraf dengan teks merah dan ukuran 14px.

Selector CSS

Selector digunakan untuk memilih elemen HTML yang ingin diberikan gaya (style). Selector bisa berdasarkan nama tag, class, atau id.

Jenis-jenis Selector:

  • p → memilih semua elemen paragraf.
  • .judul → memilih elemen dengan class="judul".
  • #utama → memilih elemen dengan id="utama".
  • * → universal selector, memilih semua elemen dalam dokumen.
  • div p → descendant selector, memilih semua paragraf di dalam elemen div.
  • div > p → child selector, memilih semua paragraf langsung di dalam elemen div.
  • p + span → adjacent sibling selector, memilih elemen span yang langsung setelah elemen p.
  • [type="text"] → attribute selector, memilih elemen berdasarkan nilai atributnya.

Contoh penggunaan:

.judul {
  color: #2563eb;
}
#utama {
  color: #16a34a;
}

Contoh HTML:

<p class="judul">Ini adalah paragraf dengan class .judul.</p>
<p id="utama">Ini adalah paragraf dengan id #utama.</p>

Ini adalah paragraf dengan class .judul.

Ini adalah paragraf dengan id #utama.

Background CSS

Contoh kode CSS untuk background:

.kotak-latar {
  background-color: lightblue;
  background-image: url('https://via.placeholder.com/300');
  background-size: cover;
  background-repeat: no-repeat;
  background-position: center;
  background-attachment: fixed;
}

Contoh HTML:

<div class="kotak-latar">
  Ini adalah kotak dengan gambar latar.
</div>
Ini adalah kotak dengan gambar latar.

Penjelasan tambahan:

  • background-position: center; → menempatkan gambar di tengah kotak.
  • background-attachment: fixed; → membuat gambar latar tetap ketika halaman digulir.

Format Teks pada Web

Contoh CSS:

h1 {
  text-align: center;
  color: navy;
  text-decoration: underline;
}

Contoh HTML:

<h1>Judul Tengah Berwarna Navy dan Garis Bawah</h1>

Judul Tengah Berwarna Navy dan Garis Bawah

Pengaturan Font

Contoh CSS untuk pengaturan font:

p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-style: italic;
  font-weight: bold;
  line-height: 1.5;
  letter-spacing: 0.5px;
}

Contoh HTML:

<p>Ini adalah paragraf dengan font Arial, italic, bold, dan pengaturan lainnya.</p>

Ini adalah paragraf dengan font Arial, italic, bold, dan pengaturan lainnya.
